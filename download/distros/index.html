Software including ViTables

  • GNU/Linux
    • openSUSE
    • Mandriva Cooker
    • Arch Linux
    • Debian/Ubuntu
    • Gentoo
    • Fink

Currently ViTables is included in several GNU/Linux distributions and it is also part of Python(x,y) a scientific-oriented Python distribution. Python(x,y) provides Windows installers for ViTables.

This page is to keep track of GNU/Linux distributions that include ViTables. If it is part of your favourite GNU/Linux distro but it is not listed in this page, just let me know and a new link will be added asap.

GNU/Linux

openSUSE

ViTables rpm packages page

Mandriva Cooker

Use the rpmfind tool to search for ViTables packages or choose a mirror from the Cooker mirrors list and go to the contrib/release section.

Arch Linux

ViTables has been included in the Arch Linux comunity-operated package repository. Its package can be found here.

Debian/Ubuntu

Packages for these distros have been gently created by Dmitrijs Ledkovs. Currently they are available here.

Gentoo

Finally ViTables can be found in the Gentoo Pkg Index. You can download it from here.

Fink

ViTables versions for Python2.5 and Python 2.6 are available. Find them using the browse packages page.
